# 🚗 Apex Motors - Responsive Landing Page

A modern and fully responsive landing page designed with a premium user experience in mind. The project features interactive navigation, smooth animations, hover effects, scroll-based interactions, and a visually appealing interface built using HTML, CSS, and JavaScript.

## 📌 Project Overview

This project was developed as part of a Web Development Internship task to demonstrate front-end development skills and modern UI/UX design principles.

The landing page includes a fixed navigation menu, responsive layouts, interactive elements, and smooth transitions to create an engaging browsing experience across all devices.

---

## ✨ Features

* Fixed Navigation Menu
* Responsive Design (Mobile, Tablet, Desktop)
* Interactive Navigation Hover Effects
* Scroll-Based Navbar Style Changes
* Smooth Scrolling Navigation
* Modern Hero Section
* Animated UI Components
* Hover Effects on Interactive Elements
* Scroll-Based Animations
* Performance Statistics Section
* Gallery Section
